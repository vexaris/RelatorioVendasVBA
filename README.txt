# Automação de Relatórios com VBA

Este projeto automatiza a criação de relatórios de vendas a partir de dados brutos em uma planilha Excel. Ele organiza os dados, gera um relatório formatado e cria um gráfico visual para facilitar a análise.

## Funcionalidades
- Extração e organização de dados de vendas
- Formatação automática do relatório
- Geração de gráficos automáticos

## Como Usar
- Abra a planilha Excel
- Pressione `Alt + F11` para abrir o editor VBA
- Cole o código na janela do editor e execute o script
